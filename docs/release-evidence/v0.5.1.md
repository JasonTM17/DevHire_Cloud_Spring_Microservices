# DevHire Cloud v0.5.1 Production Runtime Depth Evidence

## Summary

v0.5.1 raises the backend and monitoring proof after the v0.5.0 product polish baseline. The focus is production microservices depth: service integration tests, domain runtime metrics, richer demo data evidence, dashboard proof, and coverage ratchets that are backed by new tests instead of optimistic thresholds.

## Shipped Scope

- Added Testcontainers integration coverage for service-owned persistence paths in auth, company, application, notification, audit, and AI services.
- Added `scripts/migration-smoke.ps1` to validate all service-owned Flyway migrations against temporary PostgreSQL databases.
- Added domain Micrometer metrics for recruitment funnel state, notification/email delivery, audit ingestion, transactional outbox backlog, job search adapters, and AI usage.
- Added Grafana dashboards for service health, recruitment funnel, event reliability, search/AI, and database/JVM views.
- Added `scripts/runtime-observability-smoke.ps1` and `scripts/portfolio-runtime-report.ps1` for runtime metric and evidence collection.
- Expanded dashboard/frontend evidence with richer totals for jobs, candidate application status, employer pipeline, and admin audit distributions.
- Added gateway, profile mapper, and job search metric tests, then ratcheted `api-gateway`, `job-service`, and `user-service` coverage gates.
