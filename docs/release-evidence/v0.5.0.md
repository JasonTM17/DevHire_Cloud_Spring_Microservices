## Runtime Evidence Contract

`scripts/portfolio-demo-evidence.ps1` writes sanitized JSON and Markdown reports under ignored `reports/portfolio-demo-evidence/`. It can:

- verify Docker and Gateway readiness;
- run the existing runtime verification suite;
- run optional role-based performance smoke;
- capture and promote Playwright screenshots;
- re-run visual evidence audit.

No `.env`, token, report output, backup, tfstate, or cloud credential is committed.
