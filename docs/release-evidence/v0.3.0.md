- Docker runtime stack was stopped earlier to free local memory for repeated Maven verification; no volumes were removed.

- Helm CLI is not installed locally, so Helm render is documented for CI/developer machines with Helm available.
- Browser plugin was not used for Phase 76 because Playwright provided reliable local UI evidence.
- AWS is blueprint-only; no `terraform apply` was run.
