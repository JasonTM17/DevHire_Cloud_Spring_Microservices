# DevHire Cloud v1.0.0 Release Notes

## Release Theme

DevHire Cloud v1.0.0 is the production engineering portfolio release. It packages the recruitment platform as a complete reviewer experience: backend microservices, frontend product flows, deterministic demo data, runtime observability, cloud blueprint verification, security evidence, and trilingual documentation.

## Highlights

- Java 21 and Spring Boot microservices with API Gateway, auth, user, company, job, application, notification, audit, AI, and common modules.
- Service-owned PostgreSQL databases with Flyway migrations and deterministic portfolio seed data.
- Kafka eventing with transactional outbox, retry/dead-letter posture, and idempotent consumers.
- OpenSearch search path with PostgreSQL fallback.
- Claude Haiku AI assistant with cited answers, fallback mode, tool traces, safety docs, and eval scripts.
- Next.js reviewer UI with jobs, job detail, candidate, employer, admin, and assistant surfaces.
- Docker Compose full local runtime, Kubernetes raw manifests, Helm, Argo CD, External Secrets, and AWS Terraform blueprint.
- Prometheus, Grafana, Loki, Tempo, OpenTelemetry, Mailpit, SLO docs, and domain metrics.
- GitHub Actions for CI, Docker images, docs, security, CodeQL, E2E, Terraform, and governance.
- Trilingual documentation in Vietnamese, English, and Japanese.

## Honest Scope

v1.0.0 is a production-grade portfolio release. It is not a claim of live customer SaaS traffic. AWS is apply-ready blueprint evidence unless a separate credentialed cloud deployment phase is approved.

This release note is prepared ahead of the final v1.0.0 tag. The release tag should be created only after `master` is green and v1 evidence is updated from planned to released.
