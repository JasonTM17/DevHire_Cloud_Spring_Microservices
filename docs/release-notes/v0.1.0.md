# DevHire Cloud v0.1.0 Release Notes

DevHire Cloud v0.1.0 is the first portfolio release for the production-style microservices recruitment platform.

## Highlights

- Java 21, Spring Boot 3.5, Spring Cloud 2025 microservices.
- API Gateway with JWT validation, CORS, Redis rate limiting, and centralized errors.
- Auth, profile, company, job, application, notification, audit, and search workflows.
- Kafka eventing with transactional outbox and idempotent consumers.
- OpenSearch job search with PostgreSQL fallback.
- Docker Compose full local stack.
- Prometheus, Grafana, Loki, Tempo, and OpenTelemetry.
- Kubernetes, Helm, Argo CD, and AWS Terraform blueprint.
- Trivy, Gitleaks, SBOM, Dependabot, contract tests, API smoke, k6 smoke, and Playwright E2E.
- Next.js frontend portfolio UI.
- Claude Haiku AI RAG assistant with cited answers, streaming UI, fallback mode, metrics, and audit events.

## Release Verification Targets

- `mvn -T1 clean verify`
- `cd frontend && npm run typecheck && npm run build`
- `docker compose config --quiet`
- `scripts/api-smoke.ps1 -GatewayUrl http://localhost:8080`
- `scripts/e2e-smoke.ps1 -Build -KeepRunning`
