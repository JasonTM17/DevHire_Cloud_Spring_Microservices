# DevHire Cloud v0.5.1 Release Notes

Status: released

## Release Theme

`v0.5.1` is the production runtime depth release. It consolidates the cloud completion branch, runtime observability evidence, richer deterministic demo data, and backend coverage ratchets into protected `master`.

## Highlights

- Merged cloud completion and product/runtime hardening through the protected branch flow.
- Added deterministic portfolio seed data across service-owned databases.
- Added domain metrics for applications, notifications, audit, outbox, search, and AI.
- Added runtime observability smoke and portfolio runtime report scripts.
- Added Testcontainers coverage for key repositories and migration paths.
- Added v1 roadmap artifacts as future acceptance planning, not as a published v1 release.

## Honest Scope

This release is a production engineering portfolio release. It is not a claim of live customer SaaS traffic, an AWS production deployment, or an external penetration-test report. AWS remains an apply-ready blueprint until a separate credentialed cloud phase is approved.
